# Titanic Survival Prediction 🚢

## Overview

This project predicts passenger survival on the Titanic using machine learning.

## Dataset

Titanic dataset (classification problem)

## Steps

* Data cleaning
* Feature engineering
* Encoding categorical variables
* Logistic Regression model

## Results

* Accuracy: ~80%
* Improved recall for survivors using feature engineering

## Tech Stack

* Python
* Pandas
* Scikit-learn
